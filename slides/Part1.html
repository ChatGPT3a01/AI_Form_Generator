🔑 第一步：準備 Google AI API Key（30秒）

  1. 前往 Google AI Studio
  2. 登入您的 Google 帳號
  3. 點擊「Get API Key」按鈕
  4. 建立新的 API Key
  5. 複製這個 Key（格式為 AIzaSy...）
💡 免費額度充足：Gemini 每分鐘 15 次請求，每天 1500 次，對於測試和個人使用完全足夠！
Google AI Studio API Key 建立頁面

⚙️ 第二步：部署 Google Apps Script（1分鐘）

  1. 前往 Google Apps Script
  2. 點擊「新專案」，命名為「AI 問卷生成器」
  3. 將專案中的 gas_code.js 內容貼入編輯器
  4. 點擊「部署」→「新增部署作業」
  5. 選擇「網頁應用程式」，權限設為「所有人」
  6. 完成授權流程
  7. 複製部署完成的 URL
⚠️ 注意：授權時會出現「Google 尚未驗證」警告，點擊「進階」→「前往專案」→「允許」即可。

📸 GAS 部署設定畫面

部署類型選擇「網頁應用程式」，權限設為「所有人」
部署完成後複製 URL

🚀 第三步：啟動 Flask 應用（30秒）

  1. 開啟 app.py，找到第 16 行
  2. 將 GAS URL 替換為您剛才複製的部署 URL
  3. 儲存檔案
  4. 在終端機執行以下指令：
cd D:\自己架設AI_零基礎到大師\CH6_AI_生成表單系統
python app.py
Flask 應用會在 http://127.0.0.1:5000 啟動！
終端機顯示「Running on http://127.0.0.1:5000」

📋 第四步：生成第一份 AI 問卷（1分鐘）

開啟瀏覽器訪問 http://127.0.0.1:5000，填入以下資訊：

🔑 API Key

貼上第一步取得的 Gemini API Key

📂 問卷主題

輸入「顧客滿意度調查」

📊 題目數量

輸入 5

🧩 題型配比

輸入「單選3,多選1,文字1」

關鍵字：服務品質,產品滿意度,客服態度
問卷目的：了解顧客對本公司服務的整體滿意度

✅ 生成結果展示

點擊「生成問卷內容」按鈕後，幾秒鐘內就會看到結果！
