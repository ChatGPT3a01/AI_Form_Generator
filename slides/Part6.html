🏪 案例一：月度顧客滿意度調查自動化

情境

您經營一家餐廳，每個月月底需要進行顧客滿意度調查。

📊 建立參數表

用 Google Sheets 管理每月調查參數：月份、關鍵字、問卷目的

⏰ 設定觸發器

GAS 時間觸發器每月自動執行問卷生成

📧 自動通知

生成後自動發送 Email，附上問卷連結

🤖 月度自動化腳本

function monthlyAutoSurvey() {
  const sheet = SpreadsheetApp.getActiveSheet();
  const currentMonth = new Date().getMonth() + 1;
  const row = sheet.getRange(currentMonth, 1, 1, 4).getValues()[0];

  const data = {
    apiKey: PropertiesService.getScriptProperties().getProperty('GEMINI_API_KEY'),
    model: 'gemini-2.5-flash',
    category: `${currentMonth}月顧客滿意度調查`,
    questionCount: row[3],
    keywords: row[1],
    purpose: row[2],
    questionTypes: '單選3,多選1,文字1'
  };

  const aiResult = callGeminiAPI(data);
  const formResult = createGoogleForm(aiResult, data.category);

  // 記錄連結並發送通知
  sheet.getRange(currentMonth, 5).setValue(formResult.responseUrl);
  MailApp.sendEmail({
    to: 'manager@restaurant.com',
    subject: `${currentMonth}月問卷已生成`,
    body: `問卷連結：${formResult.responseUrl}`
  });
}

🌍 進階擴展：多語言問卷生成

Gemini 支援多種語言，只需在提示詞中指定：

// 在提示詞中指定語言
const prompt = `請為「${data.category}」主題生成 ${data.questionCount} 個問卷題目。
**所有題目和選項請使用 ${data.language} 語言。**
...`;

// 或生成中英雙語版本
const prompt = `請為每個題目同時提供繁體中文和 English 版本。
JSON 格式：
{
  "questions": [{
    "question": { "zh_tw": "中文題目", "en": "English question" },
    "options": { "zh_tw": ["選項1"], "en": ["Option 1"] }
  }]
}`;
