🏗️ 三層架構概覽

這個系統採用經典的三層架構，每一層有明確的職責，層與層之間通過 API 通訊。

第一層：Flask 網頁應用

使用者介面和請求分發，接收表單資料，轉發給第二層

第二層：Google Apps Script

業務邏輯處理，呼叫 AI API，建立表單，是系統核心

第三層：Google Services

Gemini AI 生成題目，Google Forms 儲存問卷

通訊協定：所有層之間都使用 HTTP/HTTPS 協定，資料格式是 JSON

🐍 Flask 層詳細設計

Flask 應用極其簡潔，只有一個 Python 檔案 app.py，不到 60 行程式碼。

路由 1：/

返回 index.html 模板，展示使用者表單介面

路由 2：/generate

接收 POST 請求，轉發到 GAS，返回生成結果

@app.route('/generate', methods=['POST'])
def generate():
    data = request.get_json()
    response = requests.post(GAS_URL, json=data)
    return jsonify(response.json())
設計哲學：Flask 是純粹的代理層，不處理 AI 邏輯，保持輕量快速

📜 Google Apps Script 層設計

GAS 層是系統的大腦，承擔最複雜的邏輯。

doPost(e)

主入口函數，接收請求並協調處理流程

callGeminiAPI(data)

呼叫 Gemini AI，生成問卷題目

createGoogleForm()

使用 FormApp API 建立 Google Forms

function doPost(e) {
  const data = JSON.parse(e.postData.contents);
  const aiResult = callGeminiAPI(data);        // AI 生成
  const formResult = createGoogleForm(aiResult); // 建立表單
  return ContentService.createTextOutput(JSON.stringify({
    status: 'success',
    ai_result: aiResult,
    form_result: formResult
  }));
}

🤔 為什麼選擇 Google Apps Script？

✅ 深度整合

與 Google Forms、Sheets、Drive 原生整合，不需要額外認證

✅ 免費使用

完全免費的雲端執行環境，無需伺服器成本

✅ 簡單部署

一鍵部署為網頁應用程式，獲得 HTTPS URL

✅ 擴展性強

可以添加時間觸發器、郵件通知等進階功能

GAS 運行在 Google 雲端，穩定可靠，是中小型專案的絕佳選擇！

🛡️ 錯誤處理機制

整個流程涉及多次網路請求，每一步都可能失敗。我們通過層層錯誤處理確保穩定性。

Flask 層

try-except 捕獲異常，返回友善錯誤訊息

GAS 層

try-catch 包裹所有邏輯，錯誤以 JSON 返回

前端

檢查 response.status，顯示適當的錯誤提示

防禦性程式設計：任何一步失敗都能給使用者明確的錯誤訊息，不會無聲掛掉！
